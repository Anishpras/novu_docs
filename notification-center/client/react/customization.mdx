## PopoverNotificationCenter Customization

### Implementing custom bell icon[](https://docs.novu.co/notification-center/react/react-components#implementing-custom-bell-icon)

It is common that you might have a special set of icons you use within your application and you will want to replace the default:Â `NotificationBell`Â coming from our library.

For this you can easily switch theÂ `NotificationBell`Â with your own bell. Just make sure you pass theÂ `unseenCount`Â param inside and use it accordingly.

```jsx
<PopoverNotificationCenter>
  {({ unseenCount }) => (
    <CustomBell
      unseenCount={unseenCount}
      colorScheme="dark"
      unseenBadgeBackgroundColor="#FFFFFF"
      unseenBadgeColor="#FF0000"
    />
  )}
</PopoverNotificationCenter>
```

### Dark mode support[](https://docs.novu.co/notification-center/react/react-components#dark-mode-support)

To support dark mode in your application the notification center component can receive aÂ `colorScheme`Â prop that can receive eitherÂ `dark`Â orÂ `light`Â mode.

```tsx
<PopoverNotificationCenter colorScheme={"dark" || "light"}>
  {({ unseenCount }) => <NotificationBell unseenCount={unseenCount} />}
</PopoverNotificationCenter>
```

### Popover positioning[](https://docs.novu.co/notification-center/react/react-components#popover-positioning)

UseÂ `position`Â  and `offset` prop to position the popover relative to the Bell icon

```jsx
<PopoverNotificationCenter position="left-start" offset={20}>
  {({ unseenCount }) => <NotificationBell unseenCount={unseenCount} />}
</PopoverNotificationCenter>
```

### Header, Footer and Empty state[](https://docs.novu.co/notification-center/react/react-components#popover-positioning)

UseÂ `header`Â  prop to customize the header of notification center

```jsx
const Header = () => {
  return <span> My custom header </span>;
};

const Footer = () => {
  return <span> My custom footer </span>;
};

const EmptyState = () => {
  return <span> My custom empty state </span>;
};

<PopoverNotificationCenter
  header={() => <Header />}
  footer={() => <Footer />}
  emptyState={<EmptyState />}
>
  {({ unseenCount }) => <NotificationBell unseenCount={unseenCount} />}
</PopoverNotificationCenter>;
```

### Notification item

Use the listItem prop to show custom notification items.

```jsx
<PopoverNotificationCenter
  colorScheme={colorScheme}
  onNotificationClick={handlerOnNotificationClick}
  onActionClick={handlerOnActionClick}
  listItem={(
    notification,
    handleActionButtonClick,
    handleNotificationClick
  ) => {
    return (
      <a
        href="/"
        onClick={(e) => {
          e.preventDefault();
          handleNotificationClick();
        }}
      >
        {notification.content}
      </a>
    );
  }}
>
  {({ unseenCount }) => {
    return <NotificationBell unseenCount={unseenCount} />;
  }}
</PopoverNotificationCenter>
```
