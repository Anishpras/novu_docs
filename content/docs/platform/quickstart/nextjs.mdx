## Quickstart

Before you can start integrating Novu into your application, you need to create a Novu account and set up a new workflow in the Novu Dashboard. This guide walks you through those steps.

<Steps>
  <CreateAccountStep />

  <Step title="Create a new Next.js application">
    You can use the following command to create a new Next.js application, or use your own and continue from the next step.

    ```package-install 
    npx create-next-app@latest
    ```
  </Step>

  <Step title="Install the Inbox package">
    ```package-install 
    npm install @novu/nextjs 
    ```
  </Step>


  <Step title="Create the Inbox component">
    To start using the Inbox component, you need to embed it in your application. 
    Import the <Method href="/docs/platform/inbox/react/components/inbox">{`<Inbox />`}</Method>
     component and pass the <Tooltip content="The application identifier is a unique identifier for your application. You can find it in the Novu Dashboard under the API keys page.">`applicationIdentifier`</Tooltip> and <Tooltip content="The subscriber ID is the unique identifier for the user in your application, typically the user's id in your database.">`subscriberId`</Tooltip> to the component.
      
    ```tsx title="components/inbox.tsx"
    'use client';
    import React from 'react';
    import { Inbox } from '@novu/nextjs';
    import { useRouter } from 'next/navigation';
    
    export function NotificationInbox() {
      const router = useRouter();
      
      return (
        <Inbox // [!code highlight]
          applicationIdentifier="YOUR_APPLICATION_IDENTIFIER" // [!code highlight]
          subscriberId="YOUR_SUBSCRIBER_ID" // [!code highlight]
          routerPush={(path: string) => router.push(path)} // [!code highlight]
        /> // [!code highlight]
      );
    }
    ```
    <SignedOut>
      <div className="text-sm text-gray-500 text-center mt-[-12px]">
        [Sign in](https://dashboard-v2.novu.co/auth/sign-up) to get your own API keys
      </div>
    </SignedOut>
  </Step>

  <Step title="Add the Inbox to your navbar">
  Now you can import the `NotificationInbox` component and add it to your navbar, it will automatically display the Bell icon, you can customize it later.
  
    ```tsx title="pages/layout.tsx"
    import { useRouter } from 'next/navigation';
    import { NotificationInbox } from "./components/inbox";

    export default function Layout() {
      return (
        <nav>
       // [!code highlight]
          <NotificationInbox /> 
        </nav>
      );
    }
    ```
  </Step>

  <Step title="Create your first subscriber">
    Novu will automatically create a subscriber when a new `subscriberId` was detected from the `Inbox` component props.
    
    ```package-install 
    npm run dev
    ```
  </Step>

  <Step title="Trigger your first notification">
    Now you have setup the Inbox component and created a subscriber, it's now time to trigger your first notification workflow.

    <Button color="primary" className="h-9">
      Visit Dashboard
    </Button>
  </Step>

</Steps>
